      Anticheat para MTA (Multi Theft Auto)

      Bem-vindo ao projeto de Anticheat para Multi Theft Auto (MTA). Este repositrio destina-se a abrigar o desenvolvimento de um sistema anticheat para melhorar a integridade e a equidade do jogo no ambiente multiplayer do MTA.

      Viso Geral

      O objetivo deste projeto criar um anticheat eficiente e robusto para o Multi Theft Auto. O anticheat visa detectar e prevenir trapaas, hacks e outras atividades fraudulentas que possam afetar negativamente a experincia de jogo dos jogadores.

      Funcionalidades

      • Screenshot: Tirar foto da tela do jogador suspeito e mostrar ela na tela do staff e salvar numa pasta dentro do mod do anticheat.

      • Discord Webhook: Este anticheat conta tambm com Logs que sero enviadas para salas do discord atravs de webhooks.

      • No AntiScreenShot: Esta funcionalidade quando ativada apenas deixa que o jogador jogue no servidor se tiver a opo Permitir o envio de capturas de tela ativado.

      • Binds: Esta funcionalidade avisa a staff que teclas que normalmente so usadas para abrir cheats foram pressionadas.

      • Anti "Backdoor": Esta funcionalidade ajuda a descobrir comandos ocultos nos mods compilados.

      • Path: Esta funcionalidade ajuda a descobrir a localizao de um certo mod no servidor.

      • Armas blacklist: Esta funcionalidade detecta o jogador que estiver com armas no permitidas no servidor e depois banindo o mesmo (exceto staff).

      • Veiculos blacklist: Esta funcionalidade detecta o jogador que estiver com veiculos no permitidos no servidor e depois banindo o mesmo (exceto staff).

      • Game Speed: Esta funcionalidade detecta se o jogador tem a velocidade do jogo acima do normal.

      • Anti Jetpack: Esta funcionalidade detecta se o jogador no faz parte da staff e se estiver usando Jetpack o mesmo ser banido.

      • Anti Fly [BETA]: Esta funcionalidade ainda no est 100% mas a funo dela detectar se o jogador est voando sem ser parte da staff do servidor.

      • RPG Fire: Caso a funcionalidade de armas blacklist no detectar outra funcionalidade entrar em ao que esta de RPG Fire consiste em o jogador que estar usando um RPG disparar ser acionado um trigger que depois ir banir o cheater.

      • Tank Fire: quase a mesma coisa que na funcionalidade acima porm se o cheater atirar de tanque.

      • Spawn VBR: Esta funcionalidade para quem usa VBR e contm a opo de spawnar tanque e hydra, esta funo analisa se o cheater possu um tanque do lado do cliente e outro veiculo do lado do servidor, resultando assim num banimento.

      • Anti Resource Stop: Esta funcionalidade evita o desligamento de mods do servidor caso o cliente desativar mods importantes ser banido do servidor.

      • Anti Aimbot [BETA]: Esta funcionalidade ainda no est 100% mas a funo dela detectar se o jogador est movendo a sua mira rpido demais como se fosse realmente um aimbot.

      • Staff serials: Esta funcionalidade quando ativa apenas deixa entrar "staffers" com o serial registado.

      • Wallhack de Staff: Esta funcionalidade est apenas presente para os staffs do servidor que ajuda a saber onde os jogadores se encontram por perto.

      • Logs de ban e unban: Esta funcionalidade enviar um log para o discord com informaes sobre qual foi o staff que deu ban ou unban a um jogador.

      • Global ban: Esta funcionalidade exclusiva do anticheat e quando ativada, jogadores que usaram cheat recentemente ao entrar do servidor sero banidos pelo sistema.

      • Anti ban: Esta funcionalidade no deixa que jogadores com serial's registados sejam banidos.

      • Update: O anticheat conta com um sistema de atualizaes.

      • Invisible Weapon [BETA]: Esta funcionalidade ainda no est 100% TESTADA mas a funo dela detectar se o jogador possu uma arma invisvel

      • Anti Fix [BETA]: Esta funcionalidade ainda no est 100% TESTADA mas a funo dela detectar se o jogador executou alguma funo de reparar o carro, e cancelar a sua funo.

      • Anti Fire-Rate: Esta funcionalidade ainda no est 100% TESTADA mas a funo dela detectar se o jogador disparou vrios tiros em segundos, resultando depois em banimento.

      • Anti VPN: Esta funcionalidade detecta se o jogador tem alguma VPN ligada, resultando depois em um Kick.

      • Fake Vehicles: Esta funcionalidade detecta se o jogador puxou algum carro por algum mod menu.

      • Warp Veh to me: Esta funcionalidade detecta se o jogador puxou algum carro existente longe de si.

      • Screenshot API: Todas as screenshots so agora armazenadas num web server via API, podendo assim enviar uma log discord com arquivo midia.

      • Anti Max Speed: Esta funcionalidade detecta se o jogador est andando a uma velocidade absurda.

      • Screenshare: Esta funcionalidade puxa um jogador para telar a sua tela. (Crditos ao zJoaoFtw_, pois foi ele que fez.)

      • Element Data: Esta funcionalidade detecta modificaes que no deveriam ocorrer nos element data dos jogadores.

      • Lua executor: Esta funcionalidade detecta qualquer injeo de cdigo LUA no lado do cliente.

      • CMD Blacklist/Block: Esta funcionalidade pune o jogador que executar um comando blacklist.

      Aviso Legal

      Este projeto desenvolvido de forma independente e no tem afiliao oficial com os desenvolvedores do Multi Theft Auto ou da srie Grand Theft Auto. O uso deste anticheat opcional e baseado na escolha dos administradores do servidor MTA.
